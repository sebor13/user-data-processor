# User Data Processor (Python Project)

## 📌 Beschreibung
Dieses Projekt ist ein einfaches Python-Programm zur Verarbeitung und Validierung von Benutzerdaten.

Es überprüft:
- Namen (nur Buchstaben, Mindestlänge 2)
- Alter (muss eine Zahl zwischen 0 und 120 sein)
- ungültige Eingaben (leere Werte, falsches Format)
- Duplikate

## ⚙️ Funktionen
- Datenbereinigung (`strip`)
- Validierung von Namen und Alter
- Trennung in gültige und ungültige Daten
- Erkennung von Duplikaten
- Ausgabe von strukturierten Listen

## 🧠 Technologien
- Python 3
- Funktionen
- Listen & Dictionaries
- String-Verarbeitung

## 🚀 Beispiel
Eingabe:
"Ivan:25, Мaria:17, Аnna:abc"

Ausgabe:
- gültige Benutzer
- ungültige Benutzer

## 🎯 Ziel
Dieses Projekt wurde als Lernprojekt erstellt, um Grundlagen von Python, Datenvalidierung und Git/GitHub zu üben.

# User Data Processor (Python Project)

## 📌 Description
This is a simple Python project for processing and validating user data.

It checks:
- names (only letters, minimum length 2)
- age (must be a number between 0 and 120)
- invalid inputs (empty strings, wrong format)
- duplicate entries

## ⚙️ Features
- data cleaning (`strip`)
- name and age validation
- separation of valid and invalid data
- duplicate detection
- structured output lists

## 🧠 Technologies
- Python 3
- functions
- lists & dictionaries
- string processing

## 🚀 Example
Input:
"Ivan:25, Maria:17, Anna:abc"

Output:
- valid users
- invalid users

## 🎯 Goal
This project was created as a learning project to practice Python basics, data validation, and Git/GitHub workflow.
